1. Информационное общество. Информационная культура
Информационное общество — теоретическая концепция постиндустриального общества; историческая фаза возможного развития цивилизации, в которой главными продуктами производства становятся информация и знания.

Информационное общество — современный этап развития цивилизации с доминирующей ролью знаний и информации, воздействием информационно-коммуникационных технологий на все сферы человеческой деятельности и общество в целом.

Информационная культура — это умение целенаправленно работать с информацией и использовать для её получения, обработки и передачи компьютерную информационную технологию, современные технические средства и методы.
2. Назовите основные требования (должностная инструкция) к профессии и профессиональные знания «Оператора ЭВМ»
Основные обязанности оператора ЭВМ:

  • Ввод информации ЭВМ технологических носителей информации и каналов связи и вывод информации из машины;
  • Передача по каналам связи, полученных на машинах расчётах для последних операции;
  • Подготовка технических носителей информации на соответствующее устройство;
  • Запись считывания и перезапись информации с носителей 1-го вида на другой;
  • Контроль технических носителей информации;
  • Обеспечение проведение вычислительного процесса, соответствии с рабочими программами;
  • Наблюдение за работой ЭВМ;
  • Установление причин сбоя в работе процессе обработки информации;
  • Запись об использования машинного времени и замеченных дефекты работы машин в журнал о отчётах машинного времени;
  • Оформление результата выполненных работах.


Профильные качества оператора ЭВМ:
  • Высокая исполнительность;
  • Аналитические способности;
  • Хорошая оперативная память;
  • Высокий уровень концентрации, распределение;
  • Коммуникальность;
  • Склонность к монотонной работе;
  • Высокий уровень развития технических способности;
  • Устойчивые зрения;
  • Владение навыками систематизации информации.
3. Типы документов, требования к их оформлению (основы делопроизводства)
Документ — это зафиксированная на материальном носителе информация в виде текста, звукозаписи или изображения с реквизитами, позволяющими её идентифицировать.

ГОСТ по делопроизводству — это свод правил по оформлению организационно-распорядительных документов.

Делопроизводство — это мероприятия по ведению, обновлению и контролю за документацией.

Существуют 3 типа документов:
  1. Входящий (поступающий в организацию);
  2. Исходящий (направленный из организации);
  3. Внутренний (документация, создаваемая и используемая работниками в управленческом процессе).

Соблюдение требований к созданию и оформлению документации необходимо:
  • В целях обеспечения юридической силы документа;
  • Для оперативной обработки документа;
  • Для оперативного поиска документа;
  • Для создания условий работы с использованием оргтехники.
4. Файловая система хранения, поиска и обработки информации. Папки, файлы (имя, тип, путь). Операции с папками в программе-оболочке Norton Commander
Файловая система предназначена для хранения в упорядоченном виде информации (данных) на внешних запоминающих устройствах. Основные функции файловой системы: выполнении операции над файлами (удаление, создание, поиск, изменение атрибутов и т.д.)

Основным средством поиcка по атрибутам файла является программа (команда) find. По умолчанию имена найденных файлов просто выводятся на стандартный вывод, но это можно переопределить и передавать список имён найденных файлов любой команде для обработки.

Обработка информации — вся совокупность операций (сбор, ввод, запись, преобразование, считывание, хранение, уничтожение, регистрация), осуществляемых с помощью технических и программных средств, включая обмен по каналам передачи данных.

Папка — это экранный объект в графических интерфейсах операционных систем и программ, дающий доступ к каталогу файловой системы.

Файл — это поименованная область диска, предназначенная для долговременного хранения логически связанной информации (данных).

Полное имя файлов состоит из имени и расширения. Имя файла может содержать максимум 255 символов, включая пробелы и расширение до 3-ёх символов. Расширения файла может отсутствовать.

Путь к файлу — это набор символов, показывающий расположение файла или каталога в файловой системе.

Операции с папками в программе Norton Commander:
  • Создание каталога (клавиша F7);
  • Переименование каталога (клавиша F6 после выделенного);
  • Копирование (клавиша F5 после выделенного);
  • Перемещение (клавиша F6 после выделенного);
  • Удаление (клавиша F8 после выделенного);
  • Произвольное выделение (клавиша Insert после установленного курсора);
  • Поиск каталогов (клавиши Alt + F5).
5. Аппаратное, программное обеспечение рабочего места специалиста
Программное обеспечение рабочего места специалиста определяет его интеллектуальные возможности, профессиональную направленность, широту и полноту осуществления функций, возможности применения различных технических устройств (блоков).

Программное обеспечение рабочего места специалиста делится на два вида:
  1. Общее программное обеспечение (ПО);
  2. Специальное ПО.

Основные элементы общего программного обеспечения обычно поставляются вместе с персональной ЭВМ. К ним относятся:
  • операционные системы и операционные оболочки;
  • программные средства ведения баз данных;
  • программные средства организации диалога;
  • программы, расширяющие возможности операционных систем.

Главное предназначение этой части программного обеспечения — управление работой процессора, организация интерфейса между пользователем, организация доступа к памяти, периферийным устройствам и сети, управление файлами, запуск прикладных программ и управление процессом их выполнения, трансляция и выполнение программ, подготовленных на алгоритмических языках.
6. Телекоммуникационные технологии. Способы передачи информации
Телекоммуникационные технологии — это передача информации с использованием электронных видов связи. В настоящее время они основываются на компьютерных технологиях.

Телекоммуникационная сеть — это набор устройств, с помощью которых отправитель передаёт сообщение получателю по каналу, используя при этом цепочку взаимосвязанных средств: терминалы, телекоммуникационные процессоры, телекоммуникационные каналы, узлы, ПО.

Терминал — это устройство ввода-вывода, используемое в телекоммуникационной сети — компьютеры, телефоны, офисное оборудование.

Телекоммуникационные процессоры поддерживают передачу и получение данных между терминалами и компьютерами.

Телекоммуникационные каналы — средства связи (провода, коаксиальные кабели, оптоволоконные кабели, микроволновые системы и системы спутниковой связи) для соединения других компонентов телекоммуникационной сети.

Узлы телекоммуникационной сети — устройства, через которые передаются и принимаются данные.

Программное обеспечение, управляющее телекоммуникациями, находится на главных компьютерах, на компьютерах, контролирующих связи, и на компьютерах конечных пользователей. Оно контролирует ввод-вывод и управляет функциями телекоммуникационной сети.

Способы передачи информации:
  • Телефонная связь;
  • Радиосвязь;
  • Спутниковая связь;
  • Интернет.
7. Автоматизированное рабочее место (состав, принципы создания)
Автоматизированное рабочее место (АРМ) – это комплекс средств вычислительной техники и программного обеспечения, располагающийся непосредственно на рабочем месте сотрудника и предназначенный для автоматизации его работы в рамках специальности.

В состав АРМ входят:
  • Источник бесперебойного электропитания, благодаря которому возможна работа при сбоях энергоснабжения;
  • Акустические колонки, посредством которых выдаются речевые сообщения об отказах устройств, и все возможные предупреждения, и подсказки, например, потеря контроля стрелки, наличие поезда на участке приближения и т.п.;
  • Принтер, позволяющий выводить на печать протоколы работы системы, устройств и персонала.

Общие принципы создания АРМ входят:
  • Системность;
  • Гибкость;
  • Устойчивость;
  • Эффективность.
8. Вычислительные сети (определение, классификация)
Компьютерная сеть — это система, обеспечивающая обмен данными между вычислительными устройствами — компьютерами, серверами, маршрутизаторами и другим оборудованием или программным обеспечением.

Классификации вычислительной сети:
  • По территориальной распространённости;
  • По архитектуре;
  • По типу сетевой топологии;
  • По типу среды передачи;
  • По функциональному назначению;
  • По скорости передачи;
  • По сетевым операционным системам;
9. Стандартные программы, входящие в состав MS Office, их назначение
Microsoft Office — офисный пакет приложений, созданных корпорацией Microsoft.

Основные популярные программы, входящие в MS Office:
  • MS Word — текстовый процессор, предназначенный для создания, просмотра, редактирования и форматирования текстов статей, деловых бумаг, а также иных документов, с локальным применением простейших форм таблично-матричных алгоритмов;
  • MS Excel — программа для работы с электронными таблицами. Она предоставляет возможности экономико-статистических расчётов, графические инструменты и язык макропрограммирования VBA;
  • MS PowerPoint — программа подготовки презентаций и просмотра презентаций, Материалы предназначены для отображения на большом экране — через проектор, либо телевизионный экран большого размера;
  • MS Access — реляционная система управления базами данных (СУБД). Имеет широкий спектр функций, включая связанные запросы, связь с внешними таблицами и базами данных.
10. Справочная система Windows. Назначение, приёмы работы
Справочная система предназначена для получения пользователем максимально точной (релевантной) информации по интересующей его/её (и ограниченной базой статей) теме.

Справочная система Windows представляет собой набор справочных файлов и мощное средство для получения консультации и поддержки по любым проблемам, которые могут возникнуть в процессе работы с системой.

Для вызова информации об основных возможностях и приёмах работы с ОС MS Windows необходимо в «Главном меню» выбрать пункт «Справка».

Окно справочной системы состоит из двух частей. В левой части выбирается интересующий пользователя раздел, а в правой отображается его содержимое. Способ поиска интересующей информации зависит от вкладки, выбранной в левой части.

Вкладки в левом панели окна:
  • Вкладка «Содержание» по своим функциональным возможностям аналогична оглавлению в книге;
  • Вкладка «Указатель» используется для поиска информации по ключевым словам и фразам;
  • Вкладка «Поиск» применяется для поиска информации по любому слову или фразе, введённому пользователем;
  • Вкладка «Избранное» предназначена для составления списка наиболее популярных разделов справки и быстрого доступа к ним.

В справочной системе широко используется механизм гиперссылок.
11. Обработка информации. Виды, краткая характеристика
Обработка информации — вся совокупность операций (сбор, ввод, запись, преобразование, считывание, хранение, удаление, регистрация), осуществляемых с помощью технических и программных средств, включая обмен по каналам передачи данных.

Виды обработки:
  • Обработка, которая приводит к получению новой информации, нового содержания знаний.
  • Обработка, которая приводит к изменению формы, но не содержания.

Кодирование — преобразование информации в символьную форму, которая удобна для её накопления, передачи, обработки и сбора. Кодирование активно используют при работе с информацией с помощью технических средств (телеграф, радио, компьютеры и т.д.).

Структурирование данных — упорядочивание информации в хранилище, классификация, каталогизация данных. Ещё один вид обработки информации – поиск в некотором хранилище информации нужных данных, которые удовлетворяют определенным условиям поиска (запросу).
12. Графические объекты в тексте. Редактирование встроенных объектов
Графические объекты — это любой нарисованный или вставленный объект, который можно редактировать и форматировать с помощью панели инструментов рисования (встроенным редактором графических объектов). Эти объекты являются частью текстового документа.

К графическим объектам в Word относятся:
  • Автофигуры;
  • Объекты Надпись;
  • Объекты WordArt.
13. Программы обработки текста. Каковы их назначения и возможности. Элементы интерфейса текстового процессора
Программа «Блокнот» на ОС Windows предназначена для управления только с содержанием текстового документа. С помощью текстового редактора можно редактировать текст, а также осуществлять простейшее форматирование шрифта. Такие программы на подобие «Блокнота» позволяют редактировать и любые другие файлы данных, имеющие текстовый формат.

Далее текстовые процессоры — это более совершённые текстовые редакторы (например, MS Word), которые позволяют управлять не только содержанием, но и оформлением текста. Эти приложения имеют широкий спектр возможностей по созданию документов (вставка списков и таблиц, средства проверки орфографии, сохранение исправлений и др.).
14. Форматирование документа в Word (нумерация страниц, колонтитулы, колонки, сноски, спецсимволы, формулы)
Форматирование документа – это изменение его внешнего вида.

Нумерация страниц — это процесс деления документа на страницы с расстановкой нумерации согласно заданным правилам и размерам. (Вставка -> Колонтитулы -> Номер страницы -> Выбрать вид).

Колонтитул — строка, расположенная на краю полосы набора и содержащая заголовок, имя автора, название произведения, части, главы, параграфа и т. д. (Вставка -> Колонтитулы -> Верхний или нижний колонтитул -> Выбрать вид).

Колонки используются для оформления текста в несколько столбцов. (Макет -> Параметры страницы -> Колонки -> Выбрать вид).

Сноска — примечание (замечание), помещаемое внизу полосы (постраничная сноска) или в конце текста (концевая сноска). (Выделить текст (слово) -> Ссылки -> Сноски -> Вставить сноску -> Написать текст).

Спецсимволы — это те символы, которые не входят в состав букв языка (языков), цифр и некоторых знаков препинания. (Вставка -> Символы -> Символ -> Выбрать символ).

Формулы — символическая запись высказывания (которое выражает логическое суждение), либо формы высказывания. (Вставка -> Символы -> Уравнение (формула)).

15. Режимы просмотра документа
В текстовом процессоре MS Word существуют следующие режимы просмотра документа:
  • Режим разметки страницы, который является основным для набора, редактирования и форматирования документов;
  • Режим Web-документа, в котором можно просматривать и редактировать документ в том виде, в каком он отображается в браузере;
  • Режим черновика, в котором форматирование текста отображается полностью.
  • Режим чтения, в котором предназначен исключительно для чтения документа с экрана – текст разбивается на сравнительно небольшие фрагменты и отображается значительно более крупным шрифтом для удобства восприятия.
  • Режим структуры, в котором позволяет проанализировать структуру сложного иерархического документа, изменить и организовать её, позволяет автоматизировано создать оглавление документа и формировать заголовки документа.
16. Назовите параметры печатной страницы, их назначение, способы установки

При наличии разрывов разделов в документе для установки параметров страницы всего документа необходимо предварительно выделять весь документ или пользоваться диалоговым окном «Параметры страницы». К диалоговому окну «Параметры страницы» можно перейти, щёлкнув по значку группы «Параметры страницы».

18. Информационные процессоры. Хранение, передача, поиск информации
Информационные процессор — это процесс сбора (приёма), передачи (обмена), хранения, обработки (преобразования) информации.

Хранение информации имеет большое значение для многократного использования информации и передачи информации во времени. Для долговременного хранения и для неоднократной дальнейшей работы с ней используются чаще всего компьютерные носители, устройства внешней памяти и др.

Передача информации — это всегда двусторонний процесс, в котором есть источник и есть приёмник информации. Передача информации необходима для её распространения. Основными устройствами для быстрой передачи информации на большие расстояния в настоящее время являются телеграф, радио, телефон, телевизионный передатчик, телекоммуникационные сети на базе вычислительных систем.

Поиск информации — процесс поиска неструктурированной документальной информации, удовлетворяющей информационные потребности, и наука об этом поиске. Поиск информации представляет собой процесс выявления в некотором множестве документов (текстов) всех тех, которые посвящены указанной теме (предмету). Процесс поиска включает последовательность операций, направленных на сбор, обработку и предоставление информации.
19. Аппаратное обеспечение: состав, назначение, обоснование выбора комплектации
Аппаратное обеспечение — электронные и механические части вычислительного устройства, входящие в состав системы или сети, исключая программное обеспечение и данные.

Материнская плата является основным компонентом компьютерной системы, платой, на которую устанавливаются или подключаются к ней остальные компоненты. Она отвечает за их взаимодействие и правильное функционирование. К материнской плате подключатся: процессор, видеокарта, звуковая карта, оперативная память, SSD диск, жестокий диск и другие аппаратные средства.

Процессор представляет собой электронную схему, которая устанавливается на устройство. Занимается выполнением кода и команд всего программного обеспечения. От центрального процессора зависит быстродействие всего ПК или мобильного устройства.

Оперативная память является буфером хранения данных, которыми пользуется процессор. Обладает высокой скоростью чтения и считывания информации, поэтому данные, которые находятся в ней и обрабатываются очень быстро. Является энергозависимым типом памяти, поэтому служит временным хранилищем для файлов, к которым нужен быстрый доступ и их обработка.

Накопители информации, которые независимы от подачи к ним тока. Используются для длительного хранения файлов, кино, музыки и другого контента. На данный момент SSD диск применяются для хранения на нем установленной операционной системы. А, жёсткий диск, для хранения различных файлов: кино, музыки, фотографий.

Видеокарта занимается обработкой графических данных, переносом машинного кода в графический — визуальный вид, который мы видим на мониторе. В новых материнских платах видеоадаптер по умолчанию встроен в материнскую плату, в некоторых случаях может быть встроен и в центральный процессор.

Блок питания обеспечивает питание всех компонентов, подключённых к материнской плате, включая её саму. При покупке нужно всегда рассчитывать мощность блока питания и потребление устройств, которые будут подключаться к своему ПК. Звуковая карта занимается обработкой звука. Чаще звуковую карту можно встретить уже по умолчанию встроенной в материнскую плату. Обычно её покупают те, кто занимается профессионально музыкой.

Клавиатура служит средством ввода информации. Может подключаться к компьютеру, ноутбуку, планшету, телевизору и другим устройствам. Содержит клавиши с цифрами и буквами, может иметь дополнительные элементы управления.

Компьютерная мышь также является средством ввода информации с помощью управления графическим указателем в системе и нажатием клавиш/кнопок на самой мышке.

Монитор является средством вывода информации. Сейчас мониторов выпускается огромное количество, все разных моделей и сделаны по различным технологиям.

Принтер является средством вывода информации в виде печати данных на листе бумаги. Сейчас выпускаются и 3Д принтеры, которые позволяют получать информацию не в графическом виде, а в виде напечатанных объектов.

Существуют и другие компоненты системного блока: CD/DVD привод, дисковод 3.5, но их редко используют в современных системных блоках, т.к. это уже устаревшие технологии. Чаще можно встретить — карт ридер.

21. Операционная система: назначение, состав, загрузка
Операционная система (ОС) — это комплекс системных и служебных программ. Операционная система является посредником между аппаратурой, программами и пользователем. ОС создаёт пользовательский интерфейс.

В состав ОС входят следующие модули:
  • Программный модуль, управляющий файловой системой;
  • Командный процессор, выполняющий команды пользователя;
  • Драйверы устройств;
  • Программные модули, обеспечивающие графический пользовательский интерфейс;
  • Сервисные программы;
  • Справочная система.

Специальная программа, содержащаяся в BIOS, начинает поиск загрузчика операционной системы. Происходит поочерёдное обращение к имеющимся в компьютере дискам (гибким, жёстким, CD-ROM) и поиск на определенном месте (в загрузочном секторе диска) наличия специальной программы Master Boot (программы-загрузчика операционной системы). Если диск системный и программа-загрузчик оказывается на месте, то она загружается в оперативную память и ей передаётся управление работой компьютера. Программа ищет файлы операционной системы на системном диске и загружает их в оперативную память в качестве программных модулей. Если системные диски в компьютере отсутствуют, на экране монитора появляется сообщение «Non system disk», и компьютер зависает, то есть загрузка операционной системы прекращается и компьютер остаётся неработоспособным. После окончания загрузки операционной системы управление передаётся командному процессору.

23. Гипертекст. Технология WWW
Гипертекст – это представление связанных ссылками фрагментов текста (вернее, множества в чём-то одинаковых по содержанию текстов) в виде сети.

Гиперссылка состоит из двух частей: указателя и адресной части. Указатель ссылки обычно выделен синим цветом и подчёркиванием. Активизация указателя гиперссылки вызывает переход на другую страницу.

Адресная часть гиперссылки представляет собой URL-адрес документа, на который указывает ссылка. Универсальный указатель ресурсов (URL) включает в себя способ доступа к документу, имя сервера, на котором находится документ, а также путь к файлу (документу).

Система World Wide Web (WWW), предназначена для поиска неуловимых данных Интернет. Для перехода из одного места поиска к другому в системе Web применяется гипертекст, который соединяет различные документы на основе заранее заданного набора слов.

Технология WWW позволяет создавать ссылки, которые реализуют переходы не только внутри исходного документа, но и на любой другой документ, находящийся на данном компьютере и на любой документ любого компьютера, подключённого к Интернету.

25. Структура программного обеспечения, назначение
Совокупность программ, предназначенная для решения задач на ПК, называется программным обеспечением.

Программное обеспечение, можно условно разделить на три категории:
  1. Системное ПО — это комплекс программ, которые обеспечивают эффективное управление компонентами компьютерной системы, выполняющие различные вспомогательные функции (проверка работоспособности устройств компьютера, выдача справочной информации и т.д.);

  3. Инструментальное ПО, обеспечивающее разработку новых программ для компьютера на языке программирования.
26. Технология хранения, поиска, сортировки данных
База данных (БД) — это организованная совокупность данных, предназначенная для длительного хранения во внешней памяти ЭВМ, постоянного обновления и использования.

Классификация баз данных возможна по характеру информации:
  • Фактографические и документальные БД;
  • Иерархические, сетевые и реляционные БД;
  • Централизованные и распределённые БД.

Фактографическая БД — БД, содержащая информацию, относящуюся непосредственно к предметной области.

В документальных БД единицей хранения является какой-либо документ, и пользователю в ответ на его запрос выдаётся либо ссылка на документ, либо сам документ, в котором он может найти интересующую его информацию.

Иерархическая модель данных — это модель данных, где используется представление базы данных в виде древовидной (иерархической) структуры, состоящей из объектов (данных) различных уровней.

Реляционные БД (РБД) — наиболее распространённый тип баз данных, использующий табличное представление данных. Системы управления базами данных (СУБД) — комплекс программных средств, предназначенных для создания структуры новой базы, наполнения её содержанием, редактирования содержимого и визуализации информации.

Основные действия, которые пользователь может выполнять с помощью СУБД:
  • создание структуры БД;
  • заполнение БД информацией;
  • изменение (редактирование) структуры и содержания БД;
  • поиск информации в БД;
  • сортировка данных;
  • защита БД;
  • проверка целостности БД.

27. Понятие баз данных (БД): состав, назначение, основные объекты
Базы данных (БД) — это взаимосвязанная информация (данные) об объектах, которая организованна специальным образом и хранится во внешней памяти компьютера.

Основные объекты БД:
  • Таблицы — служат для хранения данных в определенной структуре;
  • Запросы — создаются для выборки данных из одной или нескольких связанных таблиц;
  • Формы — предназначены для ввода, редактирования и просмотра табличных данных на экране в удобном виде;
  • Отчёты — являются выходными документами, предназначенными для вывода на принтер.

28. Выполнение работ в программах пакета MS Office
Microsoft Office — офисный пакет приложений, созданных корпорацией Microsoft.

Основные популярные программы, входящие в MS Office:
  • MS Word — текстовый процессор, предназначенный для создания, просмотра, редактирования и форматирования текстов статей, деловых бумаг, а также иных документов, с локальным применением простейших форм таблично-матричных алгоритмов;
  • MS Excel — программа для работы с электронными таблицами. Она предоставляет возможности экономико-статистических расчётов, графические инструменты и язык макропрограммирования VBA;
  • MS PowerPoint — программа подготовки презентаций и просмотра презентаций, Материалы предназначены для отображения на большом экране — через проектор, либо телевизионный экран большого размера;
  • MS Access — реляционная система управления базами данных (СУБД). Имеет широкий спектр функций, включая связанные запросы, связь с внешними таблицами и базами данных.
